Project Bio

This project was born out of a Junior Freshman Computer Science group project involving Processing visualizations and databases. Having found no up-to-date libraries that could help us implement a SQL database API for our project, we were forced to do everything from scratch. This is an effort to avoid other students going through the same troubles. Enjoy and learn!
